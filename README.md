# 📌 GabojagoUser
> 여행 일정공유 웹앱서비스 

# 1. 제작 기간 & 참여 인원
- 2023년 3월 17일 ~ 5월 2일
- 팀 프로젝트(총 5명)
- 역할 - 팀원 (DB구조설계(전체 팀원참여), 가보자고_여행정보 파트 구현)

# 2. 프로젝트 개요
여행 일정을 실시간으로 계획하고 공유하며, 다양한 여행 정보와 상품을 제공하는 웹앱서비스
- 참고한 서비스 : 핫츠고 플랜, 트립어드바이저

# 3. 사용 기술
## `Back-end`
- Java 17
- Spring Boot 3.0.5
- Gradle
- MySQL
- MyBatis
- Ajax
- Websocket
## `Front-end`
- HTML5
- CSS3
- ThymeLeaf
- JavaScript
- BootStrap
## `협업 & 개발 툴`
- IntelliJ
- github
- Figma
- diagrams.net
- ERD
